# Tableau-Project
## 📊 Health Operations Performance Dashboard | Tableau
Interactive Business Intelligence Dashboard for Healthcare Operations (2023–2024)

## 📌 Project Overview
- Healthcare organizations often manage operational data across multiple service teams, making it difficult to obtain a unified view of business performance.
- This project demonstrates how Tableau can transform fragmented operational data into a centralized dashboard, enabling leadership to monitor KPIs, service quality, revenue, and customer satisfaction from a single interactive interface.

The dashboard analyzes two years (2023–2024) of quarterly healthcare operations data covering four service teams:
- Medicare
- Commercial
- Individual Enrollment
- Billing
- 
## 🎯 Business Problem
Each healthcare operations team maintained its data independently.
As a result,
- Reports had to be manually collected from multiple teams.
- Data cleaning was repeated every time management requested updates.
- Preparing executive presentations consumed significant time.
- Decision-makers lacked a centralized performance dashboard.

## 💡 Proposed Solution
Designed an interactive Tableau dashboard that consolidates operational data into a single business intelligence solution.
The dashboard enables stakeholders to:
- Monitor operational KPIs
- Compare team performance
- Track SLA compliance
- Analyze customer satisfaction
- Monitor ticket backlog
- Evaluate revenue contribution
- Support faster business decisions
